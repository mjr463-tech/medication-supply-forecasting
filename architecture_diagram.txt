   - Polynomial regression

   - Accuracy metrics (MAPE, MAE, RMSE)

3. app/risk_engine.py
   - ShortageRiskEngine class

6. governance/

Key Algorithms:
===============

Risk Score Calculation:
- If days_of_supply < lead_time: score = 95 (CRITICAL)
- Elif days_of_supply < reorder_threshold: score = 70 (HIGH)
- Elif days_of_supply < 30: score = 40 (MEDIUM)
- Else: score = 15 (LOW)

Shortage Probability:
- Risk > 80: P(shortage) = 0.80
- Risk 60-80: P(shortage) = 0.50
- Risk 30-60: P(shortage) = 0.15
- Risk < 30: P(shortage) = 0.02

Forecast Accuracy (MAPE):
MAPE = mean(|actual - predicted| / actual) × 100%
Target: < 10% for production use
